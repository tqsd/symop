Citing SymOp
============

If you use SymOp in your research, please cite the following work:

.. code-block:: bibtex

    @article{sekavcnik2026symbolic,
        title={Symbolic Quantum State Representation and its Simulation},
        author={Sekavcnik, Simon and Noetzel, Janis},
        journal={arXiv preprint arXiv:2603.11824},
        year={2026}
    }


The paper is available at:

https://arxiv.org/abs/2603.11824


Notes
-----

SymOp is an evolving implementation of the framework introduced in the
associated publication. The software may extend or refine aspects of the
original prototype described in the paper.
